# Example Site

> One line on what makes this site memorable — the thing you'd steal for a brief.

## Palette
The 4–6 colors that actually carry the design (not every color on the page). Name the
role of each: background, ink, accent, muted.

## Type
- **Display:** the characterful face, used with restraint.
- **Body:** the workhorse.
- **Utility/mono:** captions, data, labels (if any).
- Character of the pairing in one line.

## Layout signature
The one structural move it's remembered by — split hero, oversized type, horizontal
scroll, sticky index, editorial grid, full-bleed media…

## Motion
What animates and how: scroll-velocity type, WebGL hero, page transitions, reveal
vocabulary — or "static / minimal".

## When to reach for it
The mood or brief this card fits. This is the field that makes the library useful at
build time.

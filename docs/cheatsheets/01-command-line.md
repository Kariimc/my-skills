# 🧾 Cheat Sheet #1 — Command Line Essentials (Windows / PowerShell)

> Keep this open while you work. Every professional developer uses these every
> single day. Being fast here makes you *look and feel* like a pro.

## Finding your way around
| Command | Plain English | Example |
|---|---|---|
| `pwd` | "Where am I?" (print working directory) | `pwd` |
| `ls` | "What's here?" (list files & folders) | `ls` |
| `cd <folder>` | Walk **into** a folder | `cd Desktop` |
| `cd ..` | Walk **back out** (up one level) | `cd ..` |
| `cd ~` | Jump to your home folder | `cd ~` |

## Working with files & folders
| Command | Plain English | Example |
|---|---|---|
| `mkdir <name>` | Make a new folder | `mkdir my-project` |
| `ni <name>` | Make a new empty file | `ni notes.txt` |
| `cat <file>` | Show what's inside a file | `cat notes.txt` |
| `cp <a> <b>` | Copy a file | `cp notes.txt backup.txt` |
| `mv <a> <b>` | Move **or rename** a file | `mv notes.txt todo.txt` |
| `rm <file>` | Delete a file ⚠️ (no undo!) | `rm old.txt` |

## Running Python
| Command | Plain English |
|---|---|
| `python myfile.py` | Run your Python program |
| `python` | Open live Python (type `exit()` to leave) |
| `python --version` | Check your Python version |

## Time-savers that make you look fast 🏎️
- **Tab key** — start typing a name, press Tab to auto-complete it.
- **Up arrow** — bring back your last command (no retyping).
- **`clear`** (or `cls`) — wipe the screen clean.
- **Ctrl + C** — stop a command that's running or stuck.

## 💼 Why this matters for a job
Every coding job runs on the terminal: starting apps, saving work with Git,
installing tools, deploying to the internet. Comfort here is one of the first
signs an interviewer looks for that says *"this person can actually do the
